日程調整サポートツール

課題：手作業での日程調整は面倒で時間がかかる

候補者やお客様との面談日程を調整する際、メールでのやり取りが複数回発生し、時間がかかっていませんか？ダブルブッキングのリスクも常にあります。

Before

  • メールで候補日を複数提示
  • 返信を待ってカレンダーに手動で登録
  • 前日にリマインドメールを送信

After

このツールを使えば、以下のプロセスが完全に自動化されます。

  • Googleフォームで希望日時を回答してもらうだけ
  • 回答された日時を自動でGoogleカレンダーに登録
  • 予約完了メールとリマインドメールを自動送信

ツールの機能と使い方

（ここにツールのスクリーンショットやGIF画像を挿入します）

  1. Googleフォームに必要な情報を入力し、送信します。
  2. Googleカレンダーに自動で予定が登録されます。
  3. 登録者と担当者の両方に、予約完了のメールが届きます。

使用した技術

  • Google Apps Script (GAS)
  • Google Forms
  • Google Calendar
  • Gmail

コード（一部抜粋）


// Googleフォームの回答を取得してカレンダーに登録するGASのサンプルコード
function onFormSubmit(e) {
  const formResponse = e.response;
  const itemResponses = formResponse.getItemResponses();
  
  // 回答内容を取得
  const name = itemResponses[0].getResponse();
  const email = itemResponses[1].getResponse();
  const dateTime = new Date(itemResponses[2].getResponse());

  // カレンダーにイベントを作成
  const calendar = CalendarApp.getDefaultCalendar();
  const event = calendar.createEvent(
    `面談：${name}様`,
    dateTime,
    new Date(dateTime.getTime() + 60 * 60 * 1000) // 1時間の面談
  );
  event.addGuest(email);

  // 予約完了メールを送信
  GmailApp.sendEmail(email, '面談予約完了のお知らせ', 'ご予約ありがとうございます。詳細はカレンダーをご確認ください。');
}
