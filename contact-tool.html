問い合わせLINE通知ツール

課題：フォームからの問い合わせに気づくのが遅れ、機会損失に

Webサイトの問い合わせフォームからの連絡は、メールで受信することが一般的です。しかし、他のメールに埋もれてしまい、お客様への返信が遅れてしまうことはありませんか？

Before

  • フォームからの問い合わせがメールボックスに届く
  • 担当者が定期的にメールをチェックしないと気づけない
  • 気づいた頃には、お客様の熱が冷めてしまっていることも

After

このツールを使えば、問い合わせに即座に対応できます。

  • Googleフォームに回答があった瞬間に、指定したLINEアカウントに通知が届く
  • 外出先でもスマートフォンで問い合わせ内容をすぐに確認
  • 迅速な一次対応が可能になり、顧客満足度が向上

ツールの機能と使い方

  1. お客様がWebサイトのGoogleフォームから問い合わせを送信します。
  2. ほぼリアルタイムで、あなたのLINEに問い合わせ内容が通知されます。
  3. すぐに内容を確認し、お客様への対応を開始できます。

使用した技術

  • Google Apps Script (GAS)
  • Google Forms
  • LINE Notify

コード（一部抜粋）


// Googleフォームの送信をトリガーにLINEに通知するGASのサンプルコード
function onFormSubmit(e) {
  const formResponse = e.response;
  const itemResponses = formResponse.getItemResponses();
  
  let message = '問い合わせがありました。\n';
  for (let i = 0; i < itemResponses.length; i++) {
    const question = itemResponses[i].getItem().getTitle();
    const answer = itemResponses[i].getResponse();
    message += `\n${question}: ${answer}`;
  }

  sendLineNotify(message);
}

function sendLineNotify(message) {
  const token = 'YOUR_LINE_NOTIFY_TOKEN'; // LINE Notifyのアクセストークン
  const options = {
    'method': 'post',
    'payload': `message=${message}`,
    'headers': {
      'Authorization': `Bearer ${token}`
    }
  };
  UrlFetchApp.fetch('https://notify-api.line.me/api/notify', options);
}
